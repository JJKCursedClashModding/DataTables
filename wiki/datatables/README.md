# DataTables

All `input_json` UE DataTable exports in one place.

- **Indexed** tables are split into `Name1.json` … `NameN.json` (numeric suffix only). Each group has one wiki page covering every shard.
- **Single-file** tables are one JSON per export; each has its own page.

- [Localization (text tables)](localization/) — 155 single-file
